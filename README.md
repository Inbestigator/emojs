# emojs

Simple AST and language mainly using emoji syntax

```sh
# Variables
😃👉Happy
# Logging
🗣️😃 # Prints 'Happy'
☹️👉Sad
# Functions without args
😕👉▶️❓☹️🟰Sad▶️🗣️Why so sad?
# Functions with args and multiple lines (🫷≈semicolon)
😡👉🔧🔍▶️☹️👉🔍🫷🗣️Set ☹️ to ➕☹️
# Calling functions
😕
😡😃
😕
```

You can run this exact script in `src/index.ts`
